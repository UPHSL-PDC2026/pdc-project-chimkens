## Project Overview

This project implements a parallel data processing system using Python multiprocessing to analyze a moderately large dataset. The goal of the project is to compare the performance of sequential and parallel execution using shared-memory parallelism. 

The dataset used in this project is the Global COVID-19 dataset containing more than 180,000 rows. The system loads the dataset from a CSV file and performs several data operations such as filtering, aggregation, and sorting. Two versions of the program were implemented: a sequential version and a parallel version. Execution time for both versions was measured and compared to evaluate performance improvement.

## Instructions for Running the Project

<ol>
  <li>Install required libraries</li>
    <ul><li>Make sure Python is installed, then install dependencies:</li>
      <ul>
        <li>pip install pandas numpy</li>
      </ul>
    </ul>
  <li>Place the dataset file</li>
    <ul><li>Put the dataset file in the same folder as the notebook/script:</li>
      <ul>
        <li>global_covid19_data.csv</li>
      </ul>
    </ul>
  <li>Run the notebook or script</li>
    <ul><li>Open the project in Jupyter Notebook or Google Colab and run the cells in order:</li>
      <ul>
        <li>Load dataset</li>
        <li>Sequential operations</li>
        <li>Parallel operations</li>
        <li>Time measurement</li>
        <li>Comparison table</li>
      </ul>
    </ul>
  <li>View the results</li>
    <ul><li>The program will display:</li>
      <ul>
        <li>Sequential execution time</li>
        <li>Parallel execution time</li>
        <li>Performance comparison table</li>
        <li>Speedup value</li>
      </ul>
    </ul>
  <li>Expected output</li>
    <ul>
      <li>The output should show that both versions run successfully and display the execution time comparison.</li>
      <li>Note: Parallel execution may be slower for smaller datasets due to multiprocessing overhead.</li>
    </ul>
</ol>
