## Project Overview

This project implements a parallel data processing system using Python multiprocessing to analyze a moderately large dataset. The goal of the project is to compare the performance of sequential and parallel execution using shared-memory parallelism. 

The dataset used in this project is the Global COVID-19 dataset containing more than 180,000 rows. The system loads the dataset from a CSV file and performs several data operations such as filtering, aggregation, and sorting. Two versions of the program were implemented: a sequential version and a parallel version. Execution time for both versions was measured and compared to evaluate performance improvement.
